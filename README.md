## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/pardeep1916P/Myportfolio_
cd Portfolio

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

The portfolio will be available at `http://localhost:4321`

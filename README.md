# Portfolio

Personal portfolio showcasing my cloud engineering projects and skills.

## Tech Stack

- **Astro** - Static site framework
- **React** - Interactive components  
- **TailwindCSS** - Styling
- **TypeScript** - Type safety
- **Vercel** - Deployment

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```
